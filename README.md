## 🚀 Quick Start Guide

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Git

### Running React Projects

1. Navigate to the project folder:
```bash
cd hunarmand-punjab-assignment-<number>/<project-name>
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Open browser and navigate to the local development URL (usually `http://localhost:5173`)
